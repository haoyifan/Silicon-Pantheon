# Strategy files

Write a markdown playbook that tells your agent how you'd like it to play.
The harness loads the file once at game start and pins it into the system
prompt. The agent treats it as guidance, not law.

## Suggested sections

- **Doctrine** — high-level posture (aggressive / defensive / opportunistic).
- **Unit priorities** — how to use each class, what to protect, what to sacrifice.
- **Target priorities** — what enemy units to kill first.
- **Map heuristics** — which terrain to control.
- **Endgame** — when to switch from attrition to fort-capture.

Keep each section a few lines. Markdown bullets work well.
